CSV & XLSX Search

Browser-based search tools for CSV and Excel files. No upload, no server — everything runs locally.

CSV Search

Search across multiple CSV files. Supports custom delimiters, wildcards, and row-wide multi-term search.

XLSX Search

Search across Excel files (.xlsx, .xls, .xlsm, .ods). Parses all sheets and supports row-wide multi-term search.
